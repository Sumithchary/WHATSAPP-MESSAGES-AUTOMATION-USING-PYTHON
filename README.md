# 📱 WhatsApp Automation Tool using Python (GUI-Based)

## 📌 Project Overview
This project is a **WhatsApp Automation Tool** built using **Python, Tkinter, and PyWhatKit**. It enables users to send **messages and images** to individuals, groups, or multiple contacts through an easy-to-use graphical interface.

The application supports both:
- ⚡ Instant sending  
- ⏳ Scheduled sending (time-based)  

This project demonstrates **automation, GUI development, and integration with WhatsApp Web**.

---

## 🎯 Objectives
- Automate WhatsApp messaging and image sharing  
- Provide a **simple and interactive GUI**  
- Enable **bulk messaging functionality**  
- Support **scheduled communication**  

---

## 🚀 Features

### 📩 Messaging
- Send message to a single person  
- Send message to a WhatsApp group  
- Send message to multiple contacts  
- Schedule messages  

### 🖼️ Image Sharing
- Send image to a single contact  
- Send image to a group  
- Send image to multiple contacts  
- Schedule image sending  

### 🖥️ GUI
- Menu-driven interface  
- Easy navigation  
- Error handling with alerts  
- Success notifications

## 🛠️ Tech Stack
- **Language:** Python  
- **GUI:** Tkinter

---

## ⚙️ System Workflow
1. Launch application  
2. Select:
   - Send Image / Send Message  
3. Choose:
   - Single / Group / Multiple Contacts  
4. Choose mode:
   - Instant / Scheduled  
5. Enter required details:
   - Phone number / Group ID  
   - Message / Image path  
   - Time (if scheduled)  
6. Click send → WhatsApp Web opens → Message/Image sent
